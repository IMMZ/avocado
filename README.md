# Avocado graphic engine
## Description
Avocado engine is an attempt to write a renderer on C++ using Vulkan.
Main keys are performance and ability to use in some demo game.
Written with:
- C++17
- Vulkan (GAPI)
- SDL (window management, keyboard handling)
- Catch3 (unit tests)
